Original FraCaS dataset contains 346 problems 
but some of them lack a well-defined gold answer.

FraCaS_gold.txt lists problems IDs and gold answers of 
those (334) FraCaS problems that have a well-defined gold answer. 

The IDs of the problems are preserved.

The list of omitted problems is:
[12,16,61,62,77,78,213,276,305,308,309,310]

fracas.xml represents the file downloaded from:
https://nlp.stanford.edu/~wcmac/downloads/fracas.xml
